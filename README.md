## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/guilherme.git

  # Enter folder
  $ cd guilherme

  # Install dependences
  $ yarn

  #Execute
  $ yarn dev

```
